#  E-Commerce App

A responsive frontend ecommerce interface to browse products, manage a cart, and simulate a shopping experience.

## 🚀 Live Demo
👉 https://my-ecommerce-app-phi.vercel.app/herren

## 🛠️ Tech Stack
- Next.js (App Router)
- TypeScript
- Prisma
- Tailwind CSS
- Docker
- Zustand
- Vercel

## ✨ Features
- Product listing  
- Shopping cart system  
- Dynamic UI updates (product filtering)  
- Responsive design 

## 🧠 Architecture & Implementation
- Centralized state management using Zustand to handle global state like the shopping cart  
- Client Components for interactive UI elements  
- Prisma for data modeling and database access  

## ⚙️ Challenges & Decisions

1. Cart State Management
   
   Managing the shopping cart state efficiently using Local Storage to ensure data persists across sessions.

2. Reusable Components
   
   Designing reusable product components to simplify maintenance and improve scalability.

3. Product Filtering
   
   Implementing dynamic product filters (price, color, size, brand) to allow users to filter products in real-time.
